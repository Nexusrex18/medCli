# MedCli

A command-line interface for traditional medicine terminology and codes.A **FHIR R4-compliant Traditional Medicine to ICD-11 TM2 mapping service** built with Spring Boot and MongoDB. Integrates traditional Indian medicine codes with WHO ICD-11 Traditional Medicine Module 2 for healthcare interoperability.
